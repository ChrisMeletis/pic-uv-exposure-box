# pic-uv-exposure-box
An Ultraviolet exposure box for creating PCBs (Printed Circuit Boards)
Using "Photoresist" etching method

-Microcontroller used: Pic-16F876a
  Clocked on 4Mhz
  -Peripherials:
    Text LCD monitor
    Buzzer
    Led pannel (driven with TIP-142 transistor)
